obsidian基础操作

发表于 2022-01-16 更新于 2024-05-11 分类于 5-随笔 ， 知识管理 阅读次数： Valine：
本文字数： 878 阅读时长 ≈ 1 分钟

Obsidian是基于 Markdown文件的本地强大知识管理软件，就让Obsidian作为你的第二大脑吧。

1. 基础操作

1.1 图片标准markdown语法

把使用WIKI链接关闭

1.2 源码模式切换

Obsidian 分为 阅读视图和编辑视图。在编辑视图中，还有「实时预览」、「源码模式」之分。

1.3 顶部大纲icon隐藏

其实只需要关闭核心插件的大纲即可。

1.4 搜索

首先使用obsidian自带的搜索就行，很强大。但是需要把它居中弹出显示。

使用的是 Floating Search 插件，弹出在中间，方便搜索。

2. Tab

2.1 pin tab 不要关闭

  • https://github.com/hdykokd/obsidian-advanced-close-tab

去release下载，手动安装这个插件。然后去设置快捷键cmd+w，把系统的快捷键cmd+w取消。

2.2 以新标签的形式打开文件

  • https://github.com/patleeman/obsidian-open-in-new-tab

2.3 一个tab只打开一次

  • https://github.com/scambier/obsidian-no-dupe-leaves

注意直接click，不要cmd+click

3. 主页

3.1 新标签页定制

  • https://github.com/olrenso/obsidian-home-tab

4. 表格

4.1 Advanced Tables

按如下步骤创建表格：

  1. 输入 | ;
  2. 输入列标题，然后按 Tab 键；
  3. 重复第二步，直到所有标题输入完毕，然后按 回车 键；
  4. 此时光标来到表格的第一行；
  5. 输入列的内容，然后按 Tab 键；
  6. 重复第五步，直到所有列的内容输入完毕，如果需要增加新行，按 回车 键；

4.2 Excel to Markdown Table

支持复制excel的数据，自动生成table语法
